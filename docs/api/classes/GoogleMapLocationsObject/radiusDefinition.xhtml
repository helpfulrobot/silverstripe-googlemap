GoogleMapLocationsObject::radiusDefinition

Provides MySQL snippet to work out distance between GoogleMapLocationsObject and location The method returns a string for use in queries The query snippet returns the distance between the GoogleMapLocationsObject and the latitude and longitude provided NOTE: 6378.137 is the radius of the earth in kilometers
